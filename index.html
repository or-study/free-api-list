import time, random
from openai import OpenAI, RateLimitError
client = OpenAI(
    api_key="你的 OpenCode API Key",
    base_url="https://opencode.ai/zen/v1",
)
# ⚠️ 使用 Responses API（非 Chat Completions）
def chat(model, msgs, max_retry=4):
    for i in range(max_retry):
        try:
            resp = client.responses.create(
                model=model, input=msgs
            )
            return resp.output_text
        except RateLimitError:
            if i==max_retry-1: raise
            time.sleep(2**i + random.random())
# 最稳定: deepseek-v4-flash-free
# 备用:   big-pickle / nemotron-3-super-free
print(chat("deepseek-v4-flash-free",
          [{"role":"user","content":"Hello!"}]))

❓ 常见问题 FAQ

Q: 免费额度用完了怎么办？
多数平台注册赠送的 token 有有效期，用完后需付费或等重置。建议：① 优先使用永久免费模型（智谱 GLM-4-Flash、百度 ERNIE-Speed、腾讯 hunyuan-lite）；② 多平台轮换使用；③ 火山引擎每天重置 200 万 token。
Q: 所有平台都兼容 OpenAI SDK 吗？
本页列出的 30 个平台中，29 个支持 OpenAI 兼容格式（只需改 base_url 和 api_key）。唯一例外是 Puter.js（前端 SDK 调用），但它也提供 OpenAI 兼容端点。
Q: 国内平台和海外平台怎么选？
国内平台延迟低（通常 <100ms）、中文优化好、无需翻墙，但模型选择较少。国际平台模型更丰富（含 GPT/Claude/Gemini），但需翻墙且延迟较高。建议：中文任务用国内，英文/多模态用国际。
Q: 免费层的数据会被用于训练吗？
⚠️ Google AI Studio 和 Mistral 免费层明确声明提示词可能用于模型训练。如需隐私保护，优先选择智谱、火山引擎、DeepSeek 等国内平台，或使用 Groq/Cerebras 等不用于训练的国际平台。
